Warum moo?

🌍

5 Sprachen, 1 Compiler

Schreibe Code auf Deutsch, Englisch, Spanisch, Franzoesisch oder Tuerkisch. Mische Sprachen in einer Datei. Perfekt fuer internationale Teams und den Programmierunterricht.

⚡

Nativ kompiliert

LLVM-basierter Compiler erzeugt nativen Maschinencode. Fibonacci(35) in 0.4 Sekunden. Cross-Compilation fuer ARM, RISC-V, WASM und 8 weitere Targets.

🎮

2D + 3D Grafik

Eingebaute SDL2 + OpenGL Runtime. Zeichne Rechtecke, Kreise, 3D-Wuerfel — direkt aus moo. Pong in 60 Zeilen.

🧩

Beste Features aller Sprachen

Pattern Matching (Kotlin), Lambdas (JS), Klassen + Vererbung (Python), Pipe-Operator (Elixir), Data-Klassen, f-Strings, List Comprehensions, Spread-Operator.

🔧

Systemprogrammierung

Unsafe-Bloecke, Syscalls, Raw Memory I/O, Linker-Scripts, Bare-Metal Support. Schreibe einen OS-Kernel in moo.

🛠️

Komplettes Oekosystem

LSP-Server, VS Code Extension, Formatter, Profiler, Debugger, Testsuite, Paketmanager, Web-Playground. Alles dabei.

Code-Beispiele

Pattern Matching
funktion bewerten(note):
    gib_zurück prüfe note:
        fall n wenn n >= 90: "Sehr gut"
        fall n wenn n >= 80: "Gut"
        standard: "Mangelhaft"
Lambdas + Pipe
setze zahlen auf [1, 2, 3, 4, 5]

setze ergebnis auf zahlen
    .filter((x) => x > 2)
    .map((x) => x * x)

zeige ergebnis  # [9, 16, 25]
Webserver (in moo!)
setze server auf web_server(8080)

solange wahr:
    setze req auf server.web_annehmen()
    web_antworten(req, "Hallo!", 200)
3D Grafik
setze win auf raum_erstelle("3D", 800, 600)
raum_perspektive(win, 45.0, 0.1, 100.0)

solange raum_offen(win):
    raum_löschen(win, 0, 0, 0.2)
    raum_würfel(win, 0, 0, 0, 1.0, "rot")
    raum_aktualisieren(win)
